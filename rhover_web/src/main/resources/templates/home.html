RhoVer Prototype will Include

  • At least 3 data checks
  • At least 3 studies
  • Traditional listings and interactive charts for anomalies
  • GUI-based configuration of new studies
  • GUI-based configuration of checks
  • Labeling of anomalies
  • Auditing and traceability of all changes caused by user action or data changes
  • Scheduled and on-demand execution of checks
  • Basic log-on

Out of Scope for Prototype

  • Full quiver of data checks
  • Fine-grained permissions
  • Robust error handling
  • System monitoring tools
  • User notifications
  • Polished user interface
